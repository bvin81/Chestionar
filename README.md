# Quiz Psihologie — Pregătire pentru examenul de stat

Aplicație web pentru pregătirea examenului de stat / licență în psihologie și psihoterapie. Conține 570 de întrebări grilă, bazate pe tematica oficială.

## Funcționalități

- **570 de întrebări** din domeniul psihologiei și psihoterapiei
- **Sesiuni de 50 de întrebări** selectate aleatoriu
- **Sistem de învățare adaptiv** — întrebările la care ai greșit apar mai frecvent în sesiunile următoare
- **„Nu mai arăta!"** — elimini din pool întrebările pe care le știi sigur
- **Răspunsuri greșite** — vizualizezi toate greșelile acumulate
- **Progres persistent** — istoricul se salvează în browser, nu se pierde la închidere
- **Resetare completă** — reiei de la zero oricând

## Cum se folosește

1. Deschide link-ul aplicației în browser
2. Răspunde la întrebări apăsând pe varianta corectă
3. Dacă știi sigur răspunsul, apasă **„Nu mai arăta!"** pentru a elimina întrebarea din pool
4. La finalul sesiunii (50 de întrebări) vezi scorul și poți începe o sesiune nouă
5. Progresul se salvează automat — poți închide browserul și continua mai târziu

## Note tehnice

- Aplicație web statică (HTML, CSS, JavaScript)
- Nu necesită instalare sau cont
- Datele se salvează local în browserul fiecărui utilizator (localStorage)
- Fiecare utilizator are propriul progres independent
